Breast Cancer Prediction

Predicting the likelihood of breast cancer using machine learning models

About this Project

This project aims to predict whether a patient has breast cancer based on various medical features. Using machine learning techniques, we train a model with data from breast cancer screenings to classify tumors as malignant or benign. This prediction helps doctors in early diagnosis, enabling timely intervention and better patient outcomes. Data-driven models provide a powerful tool in medical diagnostics!

We used a well-known dataset, the Wisconsin Breast Cancer Dataset (WBCD), which contains features like tumor size, shape, and texture. These features are crucial for classifying breast cancer and can significantly reduce human error in early detection.

How It Works

The model works by analyzing a patient's medical data, such as the size and shape of tumors, and categorizing them as malignant or benign. The dataset used for training contains several numerical features that help in predicting the cancerous nature of the tumor.

Explore the code for this project on GitHub!

Key Features Used for Prediction

  • Radius of tumor
  • Texture of tumor
  • Perimeter of tumor
  • Area of tumor
  • Smoothness of tumor boundary
  • Compactness of tumor
  • Concavity of tumor
  • Symmetry of tumor
